## 🚀 快速开始

> **Note**
>
> 在线安装 Qt 6.2.4，并将构建套件设置为 Qt_6_2_4_Clang_armeabi_v7a

### 环境

- Visual Studio 2019
- Windows 10
- Cmake 3.25.2
- Qt 6.2.4
- JDK 17.0.5
- NDK 21.3.6528147

### 拉取代码

```bash
git clone https://github.com/Face-recognition-punching-system/QFaceServer.git
```

### 构建

通过 Qt Creator 构建
